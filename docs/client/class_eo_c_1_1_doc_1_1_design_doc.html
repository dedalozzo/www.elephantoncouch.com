Detailed Description

A design document is a special CouchDB document where views, updates, rewrites and many others handlers are stored.

See also
http://guide.couchdb.org/editions/1/en/design.html

addHandler ( Handler\IHandler  $handler)

Adds a special handler to the design document.

This method checks the existence of the property $name, in fact a design document can have sections with multiple handlers, but in some cases there is one and only one handler per section, so that handler doesn't have a name.
