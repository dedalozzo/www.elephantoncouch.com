Returns true in case $arg is a formatted argument, false otherwise.

The argument type can be specified using the syntax %s/. The final slash is followed by the value.
%s - the argument is treated as a string.
%b - the argument is treated as a boolean.
%i - the argument is treated as an integer.
%f - the argument is treated as a float.
